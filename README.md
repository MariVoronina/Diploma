# Diploma
Выпускная квалификационная работа на тему "Инструментальное обеспечение поддержки принятия решений на примере АО "Вертекс""

## Описание
Представляет собой веб-приложение, разработанное с помощью фреймворка Dash. Позволяет работать с алгоритмом сбора данных о продукции компании и ее аналогах от других фармацевтических производителей. Кроме того, предоставляет просмотр и анализ как собранных данных, так и финансовой отчетности компании.

## Действия, необходимые для запуска веб-приложения:
- Необходимо загрузить папку WORK на диск С. Внутри нее в одной папке находится файл для заполнения базы данных на СУБД PostgreSQL, ее необходимо назвать delta. В другой папке находятся конфигурационные файлы со всей необходимой информацией для работы приложения.
- Заполнить базу данных и включить сервер для работы.
- Скачать папки Algorithm и Interface. В файле Main описан сам упомянутый выше алгоритм, а в файлах Utils и Site вспомогательные функции и метод веб-скрапинга данных соответственно. Для запуска веб-приложения необходим файл Interface_dash2, остальные - это данные и графики для его работы.
- Если что-то непонятно, то в папке Report находится текст дипломной работы, т.е. подробный отчет о действиях.
- В папке Confirmation_documents находится справка об использовании результатов дипломной работы компанией.

## Процесс работы с веб-приложением:
#### Первый раздел
Первый раздел упрощает работу с разработанным алгоритмом. Необходимо нажать на кнопку, и в базе обновятся данные о продукции и ее аналогах на текущее число. Если данные на сегодняшнее число уже есть в базе, то приложение предупредит об этом.
